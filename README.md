# Event Discovery App

Functional mobile prototype developed for Interactive Multimedia Applications 2025/2026.

## Concept

A social local event discovery app focused on spontaneous exploration of nearby experiences.

## Main Features

- Event discovery through map
- Event discovery through list
- Shake to Discover using accelerometer
- Event detail screen
- Join event interaction
- Visual and haptic feedback
- Interaction logs for usability testing

## Tech Stack

- Expo
- React Native
- React Navigation
- Expo Location
- Expo Sensors
- Expo Haptics
- AsyncStorage

## Run Locally

```bash
npm install
npx expo start
```
